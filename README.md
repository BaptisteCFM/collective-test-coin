# Collective Test

Collective Test est une plateforme comparative des 150 plus grosses cryptomonnaies

## Run back-end

step 1:
```bash
cd collective-test-back
```
step 2:

```bash
npm install
```

step 3:
running on the 4040 port
```bash
npm run dev
```

step 3bis:
if you want run test
```bash
npm run test
```
or for more option
```bash
npm run test-watch
```
## Run front-end

step 4:
```bash
cd collective-test-front
```
step 2:

```bash
npm install
```

step 3:
```bash
npm start
```
